## Quick Start

### Setup

1. Clone the repo
2. Install dependencies (see setup guides)
3. Configure .env files
4. Start backend: `cd backend && npm run dev`
5. Start frontend: `cd frontend && npm start`

### Development

- Backend: http://localhost:5000
- Frontend: Expo Dev Server
